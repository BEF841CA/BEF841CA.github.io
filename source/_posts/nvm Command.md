### [windows项目地址](https://github.com/coreybutler/nvm-windows)

查看已经安装的版本

```Bash
 nvm list
```

查看网络可以安装的版本

```Bash
 nvm list available
```

安装最新版本

```Bash
 nvm install
```

安装最新版的

```Bash
nvm install latest
```

切换使用指定的版本

```Bash
nvm use <version>
```

列出所有版本

```Bash
 nvm ls
```

显示当前版本

```Bash
 nvm current
```

给不同的版本号添加别名

```Bash
 nvm alias <name> <version>
```

删除已定义的别名

```Bash
 nvm unalias <name>
```

卸载制定的版本

```Bash
 nvm uninstall <version>
```

查看当前的版本

```Bash
 nvm version
```
