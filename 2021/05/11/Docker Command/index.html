Docker Command

Author: LiHaiyang Date: May 11, 2021  10:09:53 Category: Docker

docker

查看所有的容器

docker ps -a

所有的镜像

docker image ls

删除镜像

docker rmi xxxxx

删除容器

docker rm -f xxxxxx

设置开机自启动

docker update --restart=always xxxxxx

进入容器

docker exec -it xxxxx /bin/sh

docker-compose

启动应容器

docker-compose up

后台启动容器

docker-compose up -d

列出项目中目前的所有容器

docker-compose ps

停止容器

docker-compose stop

解决时区不同步问题

cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

date -R
